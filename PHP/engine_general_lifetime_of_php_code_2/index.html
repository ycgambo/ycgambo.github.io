发表于 2018-08-03 | 阅读次数:

在上篇文章中了解到，PHP的执行其实是分为两个部分的，先编译，再执行，op_array结构就是两者之间的桥梁。

此外，由于各个模块错综复杂，经常需要引用其他模块的信息，因此PHP使用了大量的全局变量。同时为了避免冲突，又把他们统一封装起来，提供了一些宏来访问这些全局变量。
