PHP 初探SPL(一) 数据结构

发表于 2017-11-14 | 阅读次数:

The Standard PHP Library (SPL) is a collection of interfaces and classes that are meant to solve common problems.

SPL(PHP标准库)是为解决通用问题的一组接口和类的集合。具体来说它提供了以下功能：

  • 标准数据结构
  • 迭代器以遍历对象
  • 接口、标准异常、类以支持文件操作
  • 一组像spl_autoload_register()这样的函数

SPL内置于PHP之中，并且在PHP 5.3.0之后的版本中将始终可用。
